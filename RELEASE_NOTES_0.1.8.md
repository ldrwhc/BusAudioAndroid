# BusAudioAndroid v0.1.8

## 更新内容

1. 迁移 main2.cpp 的站名多候选选择逻辑  
- 当同一站名匹配出多个音频时，可弹出候选列表进行选择。  
- 提供“试听”按钮，先试听再确认。  
- 若取消选择，自动回退到默认候选（最佳评分候选）。

2. 新增“静默合成”开关  
- 开启：多候选直接按评分自动选择，不弹窗。  
- 关闭：出现多候选时弹出选择框（可试听）。

3. 优化候选选择缓存  
- 按语言/站名/高音质/静默模式维度缓存，避免重复弹窗和重复计算。

4. 版本号更新  
- `versionCode`: 9  
- `versionName`: 0.1.8
